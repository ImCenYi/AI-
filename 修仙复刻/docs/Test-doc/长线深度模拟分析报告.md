#### 1.2.3 深渊遗宝系统

**遗宝属性计算**:
```
全属性倍率 = ∏(所有遗宝倍率)
品质基础倍率: N=1.01, R=1.02, SR=1.04, SSR=1.08, UR=1.20
等级成长: 每级+10%基础值

UR满级倍率 = 1.20 × 1.1^0 (UR不可升级) = 1.20
SSR满级倍率 = 1.08 × 1.1^2 = 1.31
SR满级倍率 = 1.04 × 1.1^4 = 1.52
R满级倍率 = 1.02 × 1.1^6 = 1.81
N满级倍率 = 1.01 × 1.1^9 = 2.36

#### 1.2.4 周天星窍系统（核心爆炸系统）

**加成计算**:
```
星域倍率 = (基础值^品质倍率和) × 周天系数

其中:
- 青龙攻击: 1.2^品质倍率和
- 朱雀生命: 1.2^品质倍率和
- 麒麟全属: 1.1^品质倍率和
- 周天系数 = 1 + (周天等级-1) × 0.5

指数加成 = 周天圆满次数 × 0.1%
```

**第一周天计算**（全5品天）:
```
青龙: 1.2^(5×5) = 1.2^25 ≈ 95.4倍
朱雀: 1.2^25 ≈ 95.4倍
麒麟: 1.1^25 ≈ 10.8倍
系数: 1.0
```

**第五周天计算**（全5品天）:
```
系数: 1 + (5-1) × 0.5 = 3.0
青龙: 1.2^25 × 3 ≈ 286倍
指数加成: 5 × 0.1% = 0.5%
```

```
