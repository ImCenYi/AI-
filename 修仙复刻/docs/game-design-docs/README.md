# 系统设计文档目录

## 前言
- 所有的系统要考虑玩家获取上限，但是需要给玩家开放无限刷的途径（可以给很少，但不能没有）


## 已完成设计

### 灵根淬炼系统
**核心体验**：部位强化 + 渡劫风险 + 累积保底  
**我们的设计**：
- 五行灵根（金木水火土）
- 渡劫觉醒（第1重天~50重天）
- 失败惩罚（天劫反噬，倒退修为）
- 天劫感悟（保底机制）
- 五行神通（不是简单加数值，是特殊效果）
- 材料：五行灵气（灵脉挂机产出）

### 本命法宝系统
**核心体验**：多属性roll + 品质分级 + 锁定 + 突破重置  
**我们的设计**：
- 3件本命法宝（元灵珠、混沌钟、天道笔）
- 5重禁制，每重一种神通（剑气/龙魂/虚空步等具体效果）
- 九品制（凡到下中上极灵玄地天）
- 悟道机制（随机roll禁制）
- 法宝升阶（法器→灵宝→法宝→...，×2指数成长）
- 材料：悟道点（通天塔+境界突破产出）

---

## 系统设计原则

1. **提取体验，而非抄袭**
   - 魂器强化 → 灵根渡劫
   - 仙兵洗练 → 法宝悟道

2. **融入修仙世界观**
   - 所有名词都是修仙化的
   - 机制包装符合修仙逻辑

3. **与现有系统联动**
   - 灵根材料 ↔ 灵脉/境界
   - 法宝材料 ↔ 通天塔/境界

4. **数值保持指数级**
   - 灵根：特殊效果百分比
   - 法宝：×2指数成长
