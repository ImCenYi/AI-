## 🎯 核心体验
> "多属性随机追求 + 品质分级 + 锁定机制 + 突破重置但数值爆炸"

## 💡 我们的呈现方式：本命法宝

**设定**：每个修士可温养数件本命法宝，法宝有多重"禁制"，每重禁制解锁一条神通  
**机制**：3件本命法宝，每件5重禁制 = 15条神通属性

### 3. 解封机制：悟道

```javascript
const ENLIGHTENMENT = {
    // 每次"悟道"会随机改变所有5重禁制的神通
    
    cost: {
        base: 1000,  // 悟道点
        growth: 1.2  // 每次×1.2
    },
    
    // 悟道结果
    result: {
        // 5重禁制，每重随机神通+随机品质
        generate() {
            return Array(5).fill(null).map(() => ({
                power: randomChoice(DIVINE_POWERS),
                quality: weightedRandom([9,8,7,6,5,4,3,2,1], [30,25,20,12,8,3,1.5,0.4,0.1])
            }));
        }
    },
    
    // 锁定机制：可锁定1-2重禁制
    lock: {
        max: 2,
        costPerLock: 500  // 每锁一重+500消耗
    }
};
```

### 4. 突破机制：法宝升阶

```javascript
const TREASURE_ASCEND = {
```

## 📦 材料：悟道点

### 获取方式：参悟天道

```javascript
const DAO_COMPREHENSION = {
    // 来源1：通天塔结算
    towerBonus: {
        // 每爬10层给一次悟道点奖励
        every10Floors: (floor) => Math.floor(floor / 10) * 100
    },
    
    // 来源2：境界突破
    realmBreakthrough: {
        // 每次突破大境界给悟道点
        reward: (realmLevel) => realmLevel * 500
    },
    
    // 来源3：每日顿悟（限时活动）
    dailyInsight: {
        // 每天特定时间可以"顿悟"一次
        timeWindow: '12:00-14:00, 20:00-22:00',
        reward: 200
    },
    
    // 来源4：消耗多余材料兑换
    exchange: {
        // 用溢出的装备/材料换悟道点
        rate: '1件橙装 = 50悟道点'
    }
};
```

## 🎯 体验总结

| 原游戏 | 我们的设计 |
|-------|-----------|
| 6个仙兵 | 3件本命法宝（更精简） |
| 5条属性 | 5重禁制（修仙化包装） |
| N/R/SR/SSR/UR | 九品制（凡到下中上极灵玄地天） |
| 攻击力+100 | 具体神通效果（剑气/龙魂/虚空步） |
| 突破+1 | 法宝升阶（法器→灵宝→法宝...） |
| ×1.1指数 | ×2指数（更激进的成长） |
| 打副本掉材料 | 通天塔+境界突破+限时顿悟 |
| 洗练石 | 悟道点 |
| 锁定券 | 悟道符 |

**核心体验一致，但完全融入我们的修仙世界观和现有系统。**
