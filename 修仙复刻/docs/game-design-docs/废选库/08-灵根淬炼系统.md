## 🎮 核心机制：灵根觉醒

### 1. 觉醒等级（1~50重天）

不同于简单的+1~+20数字，我们用**修仙化的等级**：

```
第1重天 → 第2重天 → ... → 第9重天
第1转 → 第2转 → ... → 第9转  
第1劫 → 第2劫 → ... → 第9劫
...直到50重天
```

### 2. 觉醒规则：渡劫机制

```javascript
const AWAKEN_SYSTEM = {
    // 每次觉醒都是"渡劫"
    
    // 成功率 = 基础成功率 + 天劫抗性 - 天劫难度
    // 天劫难度随等级指数增长
    calculateSuccessRate(currentLevel) {
        const base = 0.95;  // 95%基础
        const difficulty = 1 - Math.pow(0.95, currentLevel);  // 指数难度
        const resistance = this.tribulationResistance / 10000;  // 天劫抗性（积累值）
        
        return Math.max(0.01, base - difficulty + resistance);
    },
    
    // 失败惩罚：被天劫反噬
    failConsequences: {
        // 1-10重天：修为倒退1重
        // 11-20重天：修为倒退3重  
        // 21-30重天：修为归零（渡劫失败，从头再来）
        // 31+重天：灵根受损（暂时无法觉醒，需修复）
    },
    
    // 天劫抗性（保底机制）
    // 每次渡劫失败，积累"渡劫感悟"，提高下次成功率
    tribulationResistance: 0,  // 0-10000
    
    // 另一种保底：天道庇护
    // 连续失败5次，下次必成
    divineProtection: false
};
```

## 📦 材料：五行灵气

### 获取方式：灵脉修炼

不同于打副本，我们的设计是**挂机积累**+**主动修炼**：

```javascript
const SPIRIT_VEIN = {
    // 每个玩家有一条灵脉，自动产出五行灵气
    
    output: {
        // 基础产出 = 当前境界 × 灵脉等级
        base: () => game.realm.level * 10,
        
        // 可以选择专注某一行，产出+300%，其他-50%
        focus: null,  // 'metal'|'wood'|'water'|'fire'|'earth'
        
        // 或者五行平衡，全部+20%
        balance: false
    },
    
    // 主动修炼：消耗时间/资源，立即获得灵气
    activeCultivation: {
        cost: '灵石 × 100',
        reward: '灵气 × 1000',
        cooldown: '1小时'
    }
};
```

## 🎯 体验总结

| 原游戏 | 我们的设计 |
|-------|-----------|
| 魂器部位 | 五行灵根 |
| +17数字 | 第23重天（修仙化） |
| 失败掉级 | 渡劫失败，天劫反噬 |
| 幸运值 | 渡劫感悟/天道庇护 |
| 强化石 | 五行灵气（挂机+主动） |
| 保护符 | 护劫丹 |
| 打副本 | 灵脉挂机+主动修炼 |

**核心体验一致，但完全融入我们的修仙世界观。**
